# Buổi 2

 1. Câu Lệnh For

```bash
	for (bien khoi tao; dieu kien thuc thi; buoc nhay){
        cau lenh;
    }
Ví dụ
```bash
	for (int i = 1; i<=100; i++){
        cout<<i<<" Minh muon dau chuyen"<<'\n';
    }
```

 2. Vòng lặp while
  ```bash
	while( dieu khien){
        cau lenh;
    }
  ```
 Ví dụ:
 ```bash
	a=1;
    while (a!=0){
        cout<<'\n';
        cout<<"nhap du lieu: ";
        cin>>a;
    }
```
